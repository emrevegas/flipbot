# Case chest templates

PNG templates used by **Manage Emoji** on official cases (`/cases`).

| File | Theme |
|------|--------|
| `wood.png` | Classic wood |
| `ice.png` | Ice / crystal |
| `nature.png` | Moss / vines |
| `gold.png` | Golden ornate |
| `steampunk.png` | Steampunk |
| `ocean.png` | Oceanic |
| `celestial.png` | Starry / celestial |
| `gothic.png` | Gothic / dark |

Up to 4 item emojis from the case are composited inside the chest, then uploaded as a Discord **application emoji**.
